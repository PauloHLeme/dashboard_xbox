# Dashboard - Xbox GamePass
## Contexto

Projeto desenvolvido para análise exploratória de dados de assinaturas do Xbox Game Pass,

com foco em receita, planos, comportamento de assinatura e monetização adicional.



## Objetivo

Avaliar o desempenho dos planos, identificar padrões de comportamento dos assinantes

e analisar o impacto de add-ons e cupons na receita.



## Estrutura do Dashboard

1. Visão Geral de Receita

2. Análise de Planos de Assinatura

3. Tipo de Assinatura

4. Auto Renewal \& Risco

5. Add-ons \& Monetização

6. Cupons \& Impacto

7. Insights \& Recomendações



## Principais Insights

* Planos premium geram maior valor por cliente.
* Assinaturas anuais aumentam a previsibilidade da receita.
* Clientes com auto-renew concentram a maior parte do faturamento.
* Add-ons elevam significativamente o ticket médio.
* Cupons impactam o valor médio, mas não a receita total.



Assinaturas mensais sem auto-renew representam maior risco potencial.

## Limitações do Projeto
* Ausência de dados de cancelamento (churn real)
* Base de dados simplificada para fins educacionais
